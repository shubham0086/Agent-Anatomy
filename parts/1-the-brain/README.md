# Part 1 — The Brain

**The LLM, alone.** This is the part everyone already has from a chat box.

The brain can *reason* and *talk*. But on its own it has no way to act, no record
of the past, and it stops after a single response. Run the demo and watch it:

- forget your name the moment you ask for it (no **memory**)
- refuse to do `12 * 9` (no **hands**)
- never take a second step (no **loop**)

```bash
node demo.js
```

A brain alone is a chatbot. The next three parts turn it into an agent.
